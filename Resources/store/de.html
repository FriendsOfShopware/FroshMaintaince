Das Plugin unterstützt Sie mit diversen Tools bei der Wartung Ihres Shops. Bitte NICHT dauerhaft in einer Produktiv Umgebung verwenden, d.h. bei Nicht Benutzung Plugin wieder löschen. Die Tools finden sich im Backend unter Einstellungen > Wartung.


Dieses Plugin wird von @FriendsOfShopware entwickelt.
Maintainer dieses Plugins sind: David Neustadt und Alexander Wolf

Bei Fragen / Fehlern bitte ein Github Issue erstellen

Voraussetzungen

  • Shopware 5.3.4 oder höher
  • PHP 7.0

Tools

Kommandozeile

Dieses Tool fügt ein Kommandozeilen (semi-interaktive web shell) Fenster im Shopware Backend hinzu. Der Code basiert auf phpbash von Arrexel.

Datenbank Reset

Dieses Tool fügt ein Fenster im Shopware Backend hinzu, welches ermöglicht ausgewählte Module zurückzusetzen. Dabei werden die entsprechenden Daten unwiderruflich aus der Datenbank gelöscht! Der Code basiert auf den SQL Tipps & Tricks von Shopware

Module:

  • Kunden
  • Bestellungen
  • Artikel
  • Nummernkreise
  • Statistiken
  • Kategorien
  • Einkaufswelten
