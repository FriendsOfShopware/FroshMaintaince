This plugin adds several tools to simplifiy the maintenance of your shopware shop. Obviously it is highly recommended to not permanently deploy this plugin to a production environment. The tools are combined in the backend under the configuration / maintenance entry.


This plugin is part of the Github Organization @FriendsOfShopware entwickelt.
Maintainers of this plugin are: David Neustadt and Alexander Wolf

For questions or bugs please create a Github Issue

Requirements

  • Shopware 5.3.4 or higher
  • PHP 7.0

Tools

Command Line

This tool adds command line window (semi-interactive web shell) to the Shopware backend. It's based on phpbash by Arrexel.

Database reset

This tool simplifies the reset process of the database for specified modules and deletes all data of the module including the related ones. The code is based on the SQL tips & tricks from shopware.

Modules:

  • Customers
  • Orders
  • Products
  • Number ranges
  • Statistics
  • Categories
  • Emotion worlds
